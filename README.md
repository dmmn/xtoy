XtoY
====

sublime-text 2 package to duplicate lines while replacing x by y, width by height and so on...

### Usage: ###
press Ctrl+alt+d to XtoY duplicate your current line

## Features ###
```
x 		-> y
y 		-> x
width	-> height
height	-> width
top		-> bottom
bottom	-> top
left	-> right
right	-> left 
```
